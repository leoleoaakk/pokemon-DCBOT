1.寶可夢百科：https://wiki.52poke.com/

如果有查不到最新第九世代資料的問題，代表官網還沒更新，所以沒有資料  

機器人支援可用指令如下

!寶可夢指令

=>　確認機器人支援的指令

!寶可夢+名稱  
!編號+圖鑑編號  

=>　查詢寶可夢基本資料

!屬性+屬性名稱1,屬性名稱2   
!屬性+單屬性  
!屬性+寶可夢  

=>　查詢屬性相剋

!種族值+寶可夢　

=>　查詢種族值

!捕獲率+寶可夢　

=>　查詢捕獲率

!特性+特性名稱  
!特性+寶可夢  

=>　查詢特性詳細資料  

以上指令'+'符號僅是為了方便閱讀，輸入時請無視
